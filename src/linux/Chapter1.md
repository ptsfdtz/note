# linux

## 常用命令

1. 以 root 权限执行

```sh
sudo #
```

2. 文件

```sh
ls #列出当前目录下文件
pwd #输出当前目录
cd #改变目录
mkdir #新建目录
cat #显示文件内容
rm #删除目录
mv #移动目录
cp #复制目录
```

显示在终端输入内容

```sh
echo #
```

读取系统时间

```sh
date #
```

全面搜索正则表达式

```sh
grep #
```

显示命令使用手册

```sh
man #
```

安装软件包

```sh
apt-get
```

退出

```sh
exit
```

重新启动

```sh
reboot
```
