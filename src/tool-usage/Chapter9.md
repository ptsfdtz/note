# winget

window提供的包管理工具，类似于apt-get、yum等。

### 更新

1. 查看winget源

```sh
winget source list
```

2. 更新源

```sh
winget source update
```

3. 更新软件
```sh
winget upgrade --all
```
### 常用软件

winget install --id Microsoft.VisualStudioCode
