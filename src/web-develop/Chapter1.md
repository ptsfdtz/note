# mdbook

1. 安装 mdbook

[mdbook](https://github.com/rust-lang/mdBook/releases)

2. 配置 mdbook 环境

配置你的下载文件到环境变量中`Path`路径中

3. 安装 vscode 预览插件

vscode 拓展中下载

- Markdown All in One

- Markdown Preview Enhanced

4. mdbook 编译和预览

- 编译 mdbook

```sh
mdbook build
```

```sh
mdbook build -o
```

-预览 mdbook

```sh
mdbook watch -o
```

mdbook 会自动编译项目，无需手动编译
