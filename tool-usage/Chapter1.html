Powershell

一.开启 powershell

  1. win + R输入 pwsh

  2. 管理员身份运行ctrl + shift + enter

PowerShell 升级和 Windows Terminal 配置

  1. 查看 powershell 当前版本
$psversiontable
  1. 更新 powershell

查询 powershell

winget search powershell
  1. 下载微软版
winget install Microsoft.PowerShell
  1. 打开 json 文件

修改为

###
"defaultProfile": "{574e775e-4f2a-5b96-ac1e-a2962a402336}",
###
"list":
        [
            {
                "guid": "{574e775e-4f2a-5b96-ac1e-a2962a402336}",
                "hidden": false,
                "name": "PowerShell",
                "source": "Windows.Terminal.PowershellCore"
            },
        ]

二.文件相关命令

  1. 进入文件夹
cd ./
  1. 返回上级目录
cd ..
  1. 创建新文件
ni -ItemType Directory -Path <FolderPath>
  1. 删除文件
rm <FilePath>
  1. 移动文件
mv <SourcePath> <DestinationPath>
  1. 使用 vscode 打开文件
code .

三. 常用命令

环境变量

  1. 显示环境变量
gci env:
  1. 设置环境变量
$env:Path += <address>

网络配置

  1. 显示本机 ip
ipconfig
  1. 测试网络链接
ping <IP>
  1. 关闭防火墙
netsh advfirewall set allprofiles state off
  1. 显示网络统计信息
netstat -an
  1. 显示本地路由表
route print
  1. 配置防火墙规则
Get-NetFirewallRule
